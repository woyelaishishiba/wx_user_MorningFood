# wx_user_MorningFood
个人仿照美团模仿的一个简单的小程序，不需要支付，仅面向个人
通过权限鉴定对用户nickName进行认证，后台检查session的有效性。
目前还不完善，后期可能加入phone、openId、session_key的映射关系来防止接口攻击。
